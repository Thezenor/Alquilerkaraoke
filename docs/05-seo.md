# 05 — SEO

## Principios
- Mobile-first.
- Contenido útil, no generado masivamente sin control.
- Cada página con intención de búsqueda clara.
- Evitar contenido duplicado.
- Enlazado interno fuerte.
- Schema JSON-LD.
- Sitemap dinámico.
- Robots editable.
- Redirecciones 301 desde admin.
- Canonical por página.
- Hreflang ES/EN/FR.

## Admin SEO por página
- Title.
- Meta description.
- Slug.
- H1.
- H2/H3.
- Texto principal.
- FAQs.
- Schema.
- Imagen OG.
- Canonical.
- Index/noindex.
- Idioma.
- Redirecciones.
- Enlaces internos.

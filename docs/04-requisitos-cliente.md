# 04 — Área cliente

El cliente podrá:
- Ver eventos.
- Ver presupuesto.
- Aceptar presupuesto.
- Pagar reserva.
- Pagar restante.
- Descargar contrato.
- Firmar contrato.
- Descargar proforma.
- Subir documentos.
- Solicitar cambios.
- Añadir extras.
- Usar chat interno no en vivo.
- Descargar fotos/vídeos si procede.
- Crear lista de canciones.
- Compartir QR para invitados.

# 00 — Resumen del proyecto

Alquiler Karaoke será una web/plataforma profesional para captar, presupuestar, reservar y administrar eventos de karaoke y servicios audiovisuales relacionados.

Debe ser:
- mobile-first,
- moderna,
- profesional,
- SEO-first,
- multiidioma,
- administrable,
- preparada para Railway,
- preparada para IA en admin.

La web venderá principalmente karaoke, pero permitirá servicios adicionales tipo Zenor: DJ, sonido, iluminación, fotomatón, videomatón 360, gaming, pantallas y eventos Furor.

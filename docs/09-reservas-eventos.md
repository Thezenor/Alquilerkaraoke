# 09 — Reservas y eventos

## Estados
1. Solicitud recibida.
2. Presupuesto automático.
3. Pendiente revisión.
4. Pendiente pago reserva.
5. Reservado.
6. Confirmado.
7. En preparación.
8. En curso.
9. Finalizado.
10. Cancelado.
11. Bloqueado manualmente.

## Reglas
- Toda contratación queda pendiente de validación por admin.
- Admin puede bloquear fechas externas.
- Puede haber varios eventos el mismo día según recursos.
- Cliente solo consulta disponibilidad de su fecha.
- Controlar fecha, producto, técnico, colaborador, ciudad, horario y bloqueos.

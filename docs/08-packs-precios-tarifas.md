## Reglas
- Precio base normal: 4 horas.
- OkeBox: precio por día.
- Precios públicos: desde X € + IVA.
- Reserva configurable por pack.
- Fianza configurable por pack/producto.
- Extras por servicio configurables.

## Fórmula orientativa
Precio final = pack base + horas extra + provincia + kilómetros + extras + fin de semana + fecha especial + nocturnidad + tipo evento + otros suplementos - descuento + IVA.

## Admin debe permitir
- Crear packs.
- Editar precios base.
- Definir horas incluidas.
- Definir precio hora extra.
- Definir suplementos por provincia.
- Definir reglas por fecha.
- Definir fin de semana.
- Definir nocturnidad.
- Definir fianza.
- Definir reserva.
- Definir IVA.
- Definir descuentos por cliente.

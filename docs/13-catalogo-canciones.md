# 13 — Catálogo de canciones

## Importación
Formato: Excel.

Campos:
- Código.
- Título.
- Intérprete.
- Idioma.
- Estilo/género.
- Versión.
- Observaciones.

## Web pública
- Buscar por título.
- Buscar por intérprete.
- Filtrar por idioma.
- Descargar PDF completo.
- Descargar PDF por idioma.
- Descargar PDF por artista.
- Descargar PDF de búsqueda actual.

## Evento
- Cliente crea lista de canciones.
- Invitados pueden añadir canciones con QR.
- Admin puede limitar número de canciones.
- Exportar lista a PDF/Excel.

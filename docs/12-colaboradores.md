# 12 — Colaboradores

Ficha colaborador:
- Nombre fiscal/comercial.
- Persona de contacto.
- Teléfono.
- Email.
- Ciudad/provincia.
- Zonas donde trabaja.
- Servicios que realiza.
- Material/cosas que aporta.
- Precio/coste interno.
- Comisiones.
- Documentos.
- Observaciones.
- Estado activo/inactivo.
- Valoración interna.

Colaborador puede tener acceso limitado al panel si se habilita.

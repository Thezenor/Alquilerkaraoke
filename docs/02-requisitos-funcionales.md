# 02 — Requisitos funcionales

## Web pública
- Home.
- Servicios.
- Packs.
- Páginas ciudad/provincia.
- Páginas tipo evento.
- Blog.
- Catálogo canciones.
- Galerías.
- Formularios.
- Presupuesto/contratación.

## Formularios
- Pedir presupuesto.
- Contratar pack.
- Contacto general.
- Empresas/agencias.
- Colaboradores.
- Solicitar canción.
- Descargar catálogo.
- Acceso galería privada.

## Contratación

- Fecha.
- Ciudad/provincia.
- Horarios.
- Tipo evento.
- Extras.
- Cálculo presupuesto.
- Pago reserva/completo.
- Estado pendiente de validación.

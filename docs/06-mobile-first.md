# 06 — Mobile-first obligatorio

La web se diseña primero para móvil.

## Requisitos
- Menú superior siempre visible.
- CTA fijo de pedir presupuesto.
- WhatsApp visible.
- Formularios cortos y cómodos.
- Botones grandes.
- Tipografía legible.
- Carga rápida.
- Imágenes optimizadas.
- Evitar sliders pesados.
- Core Web Vitals como prioridad.
- El contenido importante debe existir también en móvil.

## Pruebas obligatorias
- iPhone ancho pequeño.
- Android ancho medio.
- Tablet.
- Escritorio.
- Lighthouse móvil.
- Navegación de formulario completa en móvil.

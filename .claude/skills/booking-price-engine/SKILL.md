# booking-price-engine

## Descripción
Construir y revisar motor de precios: pack, horas, provincia, km, extras, fecha, nocturnidad, descuento, IVA y reserva.

## Cuándo usar
Usar cuando la tarea esté relacionada con: construir y revisar motor de precios: pack, horas, provincia, km, extras, fecha, nocturnidad, descuento, iva y reserva.

## Procedimiento
1. Leer `CLAUDE.md` y `DECISIONS.md` si la tarea afecta decisiones globales.
2. Revisar los requisitos relacionados en `docs/`.
3. Implementar o proponer cambios mínimos y seguros.
4. Verificar mobile-first, SEO y seguridad si aplica.
5. Actualizar documentación cuando cambien decisiones o alcance.

# Alquiler Karaoke — paquete base para Claude Code y Antigravity

Proyecto preparado para crear desde cero una web/plataforma profesional para **www.alquilerkaraoke.com**.

Carpeta local recomendada en Windows:

```txt
C:\claude_alquilerkaraoke
```

## Objetivo
Crear una plataforma mobile-first, moderna, profesional y SEO-first para alquiler de karaoke y eventos, con:

- Web pública optimizada para SEO.
- Multiidioma mínimo: español, inglés y francés.
- Admin completo.
- Reservas y contratación online pendiente de validación por admin.
- Presupuestos automáticos configurables.
- Clientes con descuentos.
- Colaboradores.
- Contratos personalizados por servicio.
- Firma web.
- Proformas PDF.
- Pagos con tarjeta/Stripe, transferencia y Bizum manual.
- Catálogo público de canciones importado desde Excel.
- Descarga de PDF del catálogo.
- Galerías públicas y privadas por clave.
- Blog con IA desde admin.
- Optimización SEO de imágenes.
- Despliegue en Railway.

## Uso recomendado
1. Copiar esta carpeta completa en `C:\claude_alquilerkaraoke`.
2. Abrir terminal en esa carpeta.
3. Ejecutar Claude Code.
4. Pedir primero que lea `CLAUDE.md`, `DECISIONS.md`, `PROJECT_STATUS.md` y `docs/21-reglas-para-claude-antigravity.md`.
5. No empezar a programar sin validar la fase activa.
